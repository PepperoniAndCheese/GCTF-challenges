base65536 Python 2 and 3 implementation.

Installation
------------

.. sourcecode:: bash

   ~ $ python setup.py install

with pip

.. sourcecode:: bash

   ~ $ pip install base65536


Quick start
-----------

.. sourcecode:: python

   >>> import base65536
   >>> a = base65536.encode(b"Hello World")
   >>> print(a)
   驈ꍬ啯ꍲᕤ
   >>> print(base65536.decode(a))
   Hello World
